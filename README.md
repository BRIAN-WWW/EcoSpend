# EcoSpend
EcoSpend is a secure Android expense tracking app that helps you manage money, track spending, and make smarter financial decisions.

## 📱Features

- **Expense Tracking**
  - Record and categorize daily transactions
  - Maintain organized financial records

- **Smart Financial Insights**
  - Analyze spending habits
  - Receive personalized recommendations

- **Currency Conversion**
  - Real-time exchange rate integration
  - Accurate tracking of international expenses

- **Search & Filtering**
  - Quickly locate transactions
  - Filter by category

- **Privacy-First Storage**
  - Local data storage only
  - Offline accessibility and enhanced security


## 🛠️Tech Stack

- **Platform:** Android 
- **Language:** Java
- **APIs:**  
  - DeepSeek API (AI insights)  
  - Currency Exchange API  
- **Database:** Local storage (Room)
